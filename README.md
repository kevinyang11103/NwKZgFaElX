# 课程管理系统 java1184

## 项目概述

课程管理系统是一款基于B/S架构、MVC模式的Web应用程序，采用前后端分离的设计，前端使用Vue.js框架，后端采用Java语言和SpringBoot框架，数据库使用MySQL。

## 技术栈

- **前端**: Vue.js
- **后端**: Java, SpringBoot
- **数据库**: MySQL
- **开发工具**: IDEA, JDK1.8, Maven

## 系统角色

系统包含三个角色：管理员、教师和学生。

### 管理员模块

- 首页
- 系管理
- 专业管理
- 办理管理
- 学生管理
- 教师管理
- 课程管理
- 选课修改
- 管理员管理

### 教师模块

- 首页
- 授课查询
- 教师课表
- 成绩查询

### 学生模块

- 首页
- 选修课程
- 学生课程
- 课表查询
- 成绩查询
- 信息维护

## 运行环境

- **操作系统**: Windows
- **数据库**: MySQL 8.0
